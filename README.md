# Izbori Scraper

Ovaj program služi za preuzimanje podataka sa izbori.ba web stranice.

Ova specifična implementacija uzima podatke o ukupnom broju birača i broju važećih listića po opstini za lokalne izbore 2020. godine.

Program je napravljen u Python-u i koristi Selenium da preuzme podatke sa stranice. 

Ovaj program samo radi za lokalne izbore iz 2020, za druge izbore potrebno je promjeniti link i staviti odgovarajuce kodove.
